プライバシーポリシー

本プライバシーポリシーは、Veltiosoft株式会社（以下「当社」）が提供する macOS 向けローカル音声入力アプリ Cordierite および本製品サイト cordierite.veltiosoft.com（以下、総称して「本サービス」）における情報の取り扱いについて定めるものです。

運営者

Veltiosoft株式会社
Web サイト: veltiosoft.com
メール: contact@veltiosoft.com

適用範囲

本ポリシーは Cordierite macOS アプリおよび Cordierite 製品サイトに適用されます。Cordierite からリンクされる第三者の Web サイトやサービスは対象外です。

オンデバイス処理

Cordierite は Mac 上で音声を文字起こしします。音声データおよび文字起こし結果は、Apple Speech および/または Whisper（whisper.cpp）を用いて端末内で処理されます。当社のサーバーへ音声録音や文字起こし結果を送信・保存することはありません。

アプリ内に永続化しないデータ

Cordierite は、録音や文字起こし結果をアプリ内の履歴として保存しない設計です。音声入力セッションが終了すると、音声および transcript は後から参照できるアプリコンテンツとして残りません。

Mac 上のローカル保存

Cordierite は、Mac の Application Support 配下に次のデータをローカル保存する場合があります。

  • ホットキー、言語、認識エンジンなどのアプリ設定
  • ユーザーがダウンロードを選択した Whisper モデルファイル
  • ユーザー辞書の登録内容（機能提供時）

これらのデータは、アプリや関連ファイル、ダウンロード済みモデルを削除しない限り端末上に残ります。

macOS 権限

Cordierite が要求する macOS 権限は、音声入力機能の提供に必要なものに限られます。

  • マイク — 音声入力の録音
  • 入力監視 — グローバルホットキーの検出
  • アクセシビリティ — 文字起こし結果を利用中のアプリへ貼り付け

権限の状態確認はセットアップ案内のために端末内で行われ、当社サーバーへ収集されることはありません。

本 Web サイト

Cordierite 製品サイトは静的な製品ページです。現時点では、アクセス解析、広告トラッカー、類似の計測ツールは使用していません。ホスティング事業者が保持する Web サーバーまたは CDN の標準ログには、IP アドレス、ブラウザ種別、要求 URL などの技術情報が含まれる場合があります。

Pro プラン（予定）

Cordierite では将来、Pro という名称の任意の有料プランを提供する可能性があります。Pro は予定であり、現時点では提供していません。サブスクリプション等を導入する場合、決済処理は第三者の決済事業者が行うことがあり、その際の請求関連情報は当該事業者の規約および更新後の当社ポリシーに従って処理されます。Pro その他の有料機能の提供開始前に本ページを改定します。
